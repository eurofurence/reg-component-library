## Labelless

It is possible to create a checkbox without a label. This changes the structure in the DOM. The `input` will not be nested in a `label` node.

<Canvas>
	<Story name="Labelless" args={{
		showLabel: true,
	}}>
		{({ showLabel }) => <Checkbox value="example" label={showLabel ? 'Now you see me' : undefined}/>}
	</Story>
</Canvas>
